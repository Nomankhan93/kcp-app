# KCP Header Navigation Cleanup

This patch reduces the public header from many direct links into grouped navigation.

## Desktop header

Visible links:

- Home
- Introduction
- Leadership
- Services
- Contact
- Public Updates dropdown
- Citizen Services dropdown
- Staff Login dropdown

## Dropdown groups

### Public Updates

- Public Notices
- News / Updates
- Downloads / Forms

### Citizen Services

- Submit Complaint
- Track Complaint
- Apply Certificate
- Track Certificate

### Staff Login

- Admin Dashboard
- Councilor Dashboard

## Why

The previous header had too many direct links and wrapped into multiple rows. Grouping related links keeps the public website professional, cleaner on desktop, and easier to use on mobile.
